🔧 Quick Start Integration

1. Install dependency:

npm install @tanstack/react-query

2. Wrap app with QueryClientProvider (main.tsx):

<QueryClientProvider client={queryClient}>
  <App />
</QueryClientProvider>

3. Use in your component:

const { prediction, telemetry } = useCompanyIntelligence('INFY');
<HealthometerWidget prediction={prediction} />

🎯 Architecture Highlights

  • Zero Coupling: UI layer never calls APIs directly
  • Intelligent Caching: 1-min staleTime keeps data fresh, 5-min gcTime preserves memory
  • Resilient Fallback: AlphaVantage → Mock Data → Error Boundary
  • SEBI Compliance: All predictions include non-advisory disclaimers
  • Animation Polish: Framer Motion spring transitions on all visualizations
  • Type Safety: Full TypeScript coverage with zero errors

🔐 Environment Configuration

Create a .env file (optional - platform works without it):

VITE_ALPHA_VANTAGE_API_KEY=your_api_key
VITE_RAPID_API_KEY=your_rapid_api_key
VITE_RAPID_API_HOST=alpha-vantage.p.rapidapi.com

If no keys provided → Mock data mode enabled automatically
