# Mobile Testing Checklist (375px)

## Home Page
- [ ] Header sticky and accessible
- [ ] Search bar full width (with margins)
- [ ] Buttons stack vertically
- [ ] Touch targets > 44px
- [ ] No horizontal scroll
- [ ] Spacing adequate for fingers
- [ ] Bottom navigation accessible

## Stock Detail
- [ ] Back button visible
- [ ] Price large enough to read
- [ ] Score ring renders properly
- [ ] Metrics stack vertically
- [ ] Factor bars fit within viewport
- [ ] Buttons full width
- [ ] News readable

## Scanner
- [ ] Presets stack or wrap properly
- [ ] Table converts to cards (optional)
- [ ] Results scrollable vertically
- [ ] Action buttons accessible
- [ ] No horizontal table scroll

## Testing Steps
1. DevTools -> Toggle device toolbar
2. Set to iPhone 12 or 375x667
3. Verify all items above
4. Pinch to zoom (should work)
5. Test all buttons/interactions
