# StockStory — Retention System Report

This report outlines the engagement features designed to bring users back daily, weekly, and monthly.

---

## 1. Engagement Triggers

```
+------------+--------------------------------------------------------+---------------------------------------+
| Interval   | Retention Hook                                         | User Action                           |
+------------+--------------------------------------------------------+---------------------------------------+
| Daily      | • Daily Investor Brief                                 | Check watchlist movements, market     |
|            | • Live factor alerts & sentiment changes               | mood, and overnight news catalysts.   |
| Weekly     | • Sector rotation updates                              | Re-evaluate sector trends and adjust  |
|            | • Weekly opportunity alerts                            | stock focus lists.                    |
| Monthly    | • Monthly Portfolio Health audit                       | Review long-term target drift and     |
|            | • Corporate earnings releases & factors updates        | rebalance weights.                    |
+------------+--------------------------------------------------------+---------------------------------------+
```

---

## 2. Retention Loops

* **Daily Loop (High-Frequency Catalyst Checks)**:
  * Pushing news notifications and factor triggers (e.g. "RSI crossing oversold boundaries") keeps active traders engaged.
* **Weekly Loop (Medium-Frequency Sector Rotations)**:
  * Since sectors rotation signals (BULL, BEAR, ACCUMULATION) update weekly, users return to check if they need to shift capital.
* **Monthly Loop (Low-Frequency Portfolio Rebalancing)**:
  * Highlighting portfolio drift and risk concentration changes prompts users to run a monthly portfolio health audit.
