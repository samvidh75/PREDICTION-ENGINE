## KEY CONCEPTS

### What is SGLang?
- Framework for structured AI generation
- Works with any LLM (Ollama, vLLM, etc.)
- Guarantees JSON output
- Token-efficient
- Perfect for stock analysis

### What is Ollama?
- Local LLM runtime
- Runs Mistral-7B model
- No GPU required
- ~4-8GB RAM needed
- Free & open-source

### How they work together?
```
Ollama (LLM Runtime)
    ↑
SGLang (Structured Generation)
    ↑
Your Backend API
    ↑
Frontend UI
```
